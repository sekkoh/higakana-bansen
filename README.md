# 東神奈川到着番線　〜ひがかな番線〜

横浜駅に行くのに、なぜか東神奈川駅で降ろされ、挙げ句の果てに階段を登り降りさせられ、イラっときたことありませんか？？
このアプリは、横浜線で次の列車の東神奈川駅到着番線が一目でわかるアプリです。

## 機能
- 横浜線の乗車駅から東神奈川駅の到着番線が分かる
- 京浜東北線に乗り換えて（直通で）横浜駅の到着時刻が分かる（将来機能）
- 新子安方面への乗り継ぎ列車の時刻（将来機能）

## できないこと
- 横浜駅から先の時刻
- 新子安方面の到着時刻

## ファイルリスト（一部）
- `public/data/timetable.csv` 時刻表データ
- `scripts/generateTimeable.js` デバッグ用時刻表データ生成スクリプト
- `src/` ソースコード
